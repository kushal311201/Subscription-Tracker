Airtable Setup Guide

Complete step-by-step instructions to set up Airtable for your Subscription Manager app

What is Airtable?

Airtable is a cloud-based database service that allows you to store and organize your data. The Subscription Manager app uses Airtable to store all your subscription information securely in the cloud, which means you can access it from any device.

The free plan of Airtable is sufficient for personal use of the Subscription Manager app.

Step-by-Step Setup

1
Create an Airtable Account
  1. Go to https://airtable.com/signup
  2. Sign up with your email, or use Google/Apple/Facebook authentication
  3. Fill in the required information and complete the signup process
Note: If you already have an Airtable account, you can skip this step and simply log in.
2
Create a New Base
  1. After logging in, you'll see your workspace dashboard
  2. Click the "+ Create a base" button
  3. Select "Start from scratch"
  4. Name your base "Subscription Manager" (or any name you prefer)
3
Set Up the Subscriptions Table
  1. Once your base is created, click on "Table 1" at the top and rename it to "Subscriptions"
  2. By default, Airtable creates a table with a "Name" field. Leave that as is.
  3. Next, click the "+" button to the right of the "Name" field to add more fields
  4. Create the following fields:
Field Name Field Type Options
name Single line text Default (rename from "Name" if needed)
amount Number Format as currency (â‚¹)
billingCycle Single select Add options: monthly, yearly, quarterly
dueDate Date Default settings
category Single select Add options: streaming, fitness, productivity, music, cloud, other
Important: Field names are case-sensitive. Make sure to use exactly the field names as shown above.
4
Get Your API Key
  1. Click on your account icon in the top right corner
  2. Select "Account" from the dropdown menu
  3. Scroll down to the "API" section
  4. If you don't already have an API key, click "Generate API key"
  5. Copy your API key and save it somewhere secure (you'll need it for the app)
Security note: Your API key provides access to your Airtable data. Never share it publicly or include it in public repositories.
5
Get Your Base ID
  1. Go to https://airtable.com/api
  2. Click on the "Subscription Manager" base (or whatever you named it)
  3. Look for text that says "The ID of this base is..." and copy the Base ID (it starts with "app...")
6
Configure Your App
  1. Open the js/app.js file in a text editor
  2. Look for these lines near the top of the file:
const airtableApiKey = 'YOUR_API_KEY';
const airtableBaseId = 'YOUR_BASE_ID';
const airtableTableName = 'Subscriptions';
  1. Replace 'YOUR_API_KEY' with your actual API key (keep the quotes)
  2. Replace 'YOUR_BASE_ID' with your actual Base ID (keep the quotes)
  3. Save the file

Testing Your Configuration

Once you've completed all the steps above, you can test your configuration:

  1. Open the Subscription Manager app in your browser
  2. Try adding a new subscription using the form
  3. If everything is configured correctly, the subscription should be saved to Airtable, and you should see it appear in the "Your Subscriptions" section
  4. You can also check your Airtable base to confirm the data was saved
Troubleshooting: If you encounter any errors, check the browser console (F12 > Console) for error messages. Common issues include incorrect API key or Base ID, or field names not matching exactly.
